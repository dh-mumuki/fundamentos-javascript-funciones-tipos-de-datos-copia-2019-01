¿Y que pasa si sumo números con textos?

```javascript
var calle = "Monroe";
var numero = 860;
console.log(calle + " " + numero) 
// El resultado sería: "Monroe 860"
```
Como habrás visto, si sumo un número y un texto, éste se convierte en parte del _string_.
